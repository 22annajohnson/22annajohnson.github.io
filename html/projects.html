German Learning App

I am currently taking a beginner German course and needed some inspiration to study. So I started making an iOS app to practice my vocab. My main focuses in this app are learning CoreData, reading data from JSON files and dynamically adding UIViews and UIViewControllers.

This may not have been the most efficient way to study German, but it certainly taught me a lot about iOS development. The vocabulary pages (Volkabeln) are controlled by a UICollectionViewController to allow for dynamically added vocabulary views and side scrolling behavior. The individual vocabulary cells are contained inside a UITableView and are added dynamically from the JSON file.

Pilot Blackbox App

For my senior capstone project, my team has been tasked with creating a device to record and display real-time biometrics for a pilot during flight. This device is meant to be both a preventative and diagnostic device for aviation accidents. Data stored may be analyzed post accident to better understand the cause. This project is ongoing, and expected to be completed before graduation in May 2023.

The current goal is to design a device that attaches to the headset and send the data via Bluetooth to an app using CoreBluetooth. This app will store and display the vitals, as well as flag any abnormalities in these values. The iOS app will then interface with a WatchOS app to alert the pilot of any flags.

Cabin conditions will also be recorded and logged. This will allow the program to notify the pilot if carbon monoxide levels are too high or if cabin temperatures are too low.

Handet

My team was tasked with designing and building a prosthetic hand prototype that completed a certain task. So we decided to build Handet, the prostetic hand that can play the trumpet. The goal of this project was to explore ways in which engineering can improve quality of life as well as technical writing, design, presentation, and testing.

The prosthetic was 3D printed using PLA for the casing and TPU for the fingers. The flexible properties of TPU allowed for a flexible joint rather than a mechanical one, simplifying the actuation mechanism. Each finger was connected to a solenoid which were controlled by a Raspberry Pi 4b+. This interfaced with a GUI that allowed the user to change the input parameters, upload songs, and calibrate the device.

Two iterations of this prototype were completed: V1 and V2. Check out the Final Report V1, Proposal V1, Final Report V2, and Final Presentation V2.

Microcontroller Volume Control

A friend of mine built a computer in a record player and asked me to make a device to control the volume with the original analog knobs. I used a RaspberryPi Pico as the microcontoller and two rotary encoders. The knobs may be rotated or clicked, both with different functionalities. The code was written so that either action can equate to any series of keypresses.
